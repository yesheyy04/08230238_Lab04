My Learning Journey in Web Development

My learning journey started with the basics of programming in Python. Over time, I explored how Python can be used to build real-world applications. When I learned about web development, I discovered Django, a Python framework that makes creating websites faster and easier.

In this project, I have been learning how to:

  • Create a Django project and apps
  • Use the Model-View-Template (MVT) architecture
  • Design HTML templates for web pages
  • Connect URLs and views to render dynamic content
  • Use Django's admin interface to manage data

This hands-on experience has helped me understand how web applications work behind the scenes and how front-end and back-end components interact.
